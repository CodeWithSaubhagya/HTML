# HTML Practice Files

This folder contains simple HTML practice pages for learning common HTML elements and page structure.

## Files

- `index.html` - Basic heading and paragraph examples.
- `a.html` - Anchor/link practice.
- `div.html` - Basic `div` element practice.
- `form.html` - Form, label, input, and button examples.
- `nav.html` - Simple navigation list example.
- `small.html` - Small text element practice.
- `ul-li.html` - Unordered list and list item practice.

## How to use

Open any `.html` file in a web browser to view the page. You can edit the files in any code editor and refresh the browser to see your changes.
